# THE YON - Slot Image IDs Reference

Complete reference for all CMS-managed slot images across the website.

## Naming Convention

```
{page}-{section}-{number}
```

- **page**: Page identifier (home, about, collections, etc.)
- **section**: Section within the page (hero, philosophy, etc.)
- **number**: 3-digit number (001, 002, 003...)

## How to Add Images in Sanity Studio

1. Go to **Slot Images** in the sidebar
2. Select the page you want to edit
3. Click **+ Create** to add a new slot image
4. Enter the **Slot ID** exactly as shown above
5. Select the **Page** and **Section**
6. Upload your **Image**
7. Add **Alt Text** for accessibility
8. Set **Active** to true to display

### Tips
- Use the **order** field to control display sequence within a section
- **Label Override** changes the text displayed on the slot
- Disable slots by setting **Active** to false (keeps the image for later)
- Images automatically show with LQIP blur-up loading

## Technical Integration

Slots are fetched server-side with 60-second caching:

```typescript
// In page component
const slotImages = await getSlotImages('home')
const slot = createSlotHelper(slotImages)

// Usage
<Slot
  {...slot('home-hero-001', 'DEFAULT LABEL')}
  size="hero"
  rotation={-2}
/>
```

The `slot()` helper returns:
- `imageUrl`: CMS image URL (if uploaded)
- `imageAlt`: Alt text
- `imageLqip`: Low quality image placeholder
- `label`: Custom label or default
